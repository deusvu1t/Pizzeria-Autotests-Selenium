## Промокоды

### TC-1: Применение валидного промокода (10% скидка)

**Preconditions:**

- В корзине есть хотя бы один товар

**Steps:**

1. Перейти в корзину
2. Перейти к оформлению заказа
3. Запомнить итоговую сумму заказа
4. Ввести промокод `GIVEMEHALYAVA`
5. Применить промокод

**Expected result:**

- Промокод успешно применён
- Итоговая сумма уменьшена на 10%
- Отображается информация о применённой скидке

---

### TC-2: Применение невалидного промокода

**Preconditions:**

- В корзине есть хотя бы один товар

**Steps:**

1. Перейти в корзину
2. Перейти к оформлению заказа
3. Запомнить итоговую сумму заказа
4. Ввести промокод `DC120`
5. Применить промокод

**Expected result:**

- Промокод не применён
- Итоговая сумма не изменилась
- Отображается сообщение об ошибке (невалидный промокод)

---

### TC-3: Ошибка сервера при применении промокода

**Preconditions:**

- В корзине есть товары
- Настроен перехват запроса (mock / intercept)

**Steps:**

1. Перейти в корзину
2. Перейти к оформлению заказа
3. Ввести промокод `GIVEMEHALYAVA`
4. Перехватить запрос валидации промокода
5. Вернуть ошибку сервера (например, HTTP 500)
6. Применить промокод

**Expected result:**

- Промокод не применён
- Итоговая сумма не изменилась
- Отображается сообщение об ошибке (например: "Не удалось применить промокод")
- Приложение не падает, UI остаётся стабильным

---

### TC-4: Повторное использование промокода одним пользователем

**Preconditions:**

- Пользователь зарегистрирован и авторизован

**Steps:**

1. Добавить товар в корзину
2. Перейти к оформлению заказа
3. Применить промокод `GIVEMEHALYAVA`
4. Завершить оформление заказа
5. Создать новый заказ (добавить товар в корзину)
6. Перейти к оформлению
7. Повторно применить промокод `GIVEMEHALYAVA`

**Expected result:**

- При первом заказе промокод применяется
- При втором заказе промокод не применяется
- Отображается сообщение о том, что промокод уже использован
- Итоговая сумма второго заказа не уменьшается
