# Тест-кейсы: основной пользовательский флоу

## TC-001: Отображение слайдера с пиццами на главной странице

**Шаги:**

1. Открыть главную страницу сайта.
2. Найти слайдер с пиццами.
3. Получить список видимых слайдов.

**Ожидаемый результат:**

- Главная страница открыта.
- Слайдер с пиццами отображается.
- В слайдере отображаются карточки пицц.
- У каждой видимой карточки есть название и цена.

## TC-002: Отображение кнопки добавления в корзину после наведения на пиццу

**Шаги:**

1. Открыть главную страницу сайта.
2. Найти слайдер с пиццами.
3. Выбрать видимую пиццу в слайдере.
4. Навести курсор на карточку пиццы.

**Ожидаемый результат:**

- После наведения на карточке отображается кнопка `В корзину`.

## TC-003: Добавление одной пиццы из слайдера в корзину


**Шаги:**

1. Открыть главную страницу сайта.
2. Найти слайдер с пиццами.
3. Выбрать видимую пиццу.
4. Сохранить название и цену выбранной пиццы.
5. Навести курсор на карточку пиццы.
6. Нажать кнопку `В корзину`.
7. Открыть корзину.

**Ожидаемый результат:**

- Выбранная пицца добавлена в корзину.

## TC-004: Добавление нескольких пицц из слайдера в корзину

**Шаги:**

1. Открыть главную страницу сайта.
2. Найти слайдер с пиццами.
3. Добавить первую видимую пиццу в корзину.
4. Добавить вторую видимую пиццу в корзину.
5. Открыть корзину.

**Ожидаемый результат:**

- Выбранные пиццы добавлена в корзину.

## TC-005: Переключение слайдера вправо и влево

**Шаги:**

1. Открыть главную страницу сайта.
2. Найти слайдер с пиццами.
3. Сохранить названия видимых пицц.
4. Переключить слайдер вправо.
5. Дождаться завершения анимации.
6. Сохранить названия видимых пицц после переключения.
7. Переключить слайдер влево.
8. Дождаться завершения анимации.
9. Сохранить названия видимых пицц после обратного переключения.

**Ожидаемый результат:**

- После переключения вправо набор видимых пицц изменился.
- После переключения влево пользователь снова видит предыдущие пиццы.

## TC-006: Добавление пиццы после переключения слайдера

**Шаги:**

1. Открыть главную страницу сайта.
2. Найти слайдер с пиццами.
3. Переключить слайдер вправо.
4. Дождаться завершения анимации.
5. Выбрать пиццу из нового набора видимых слайдов.
6. Навести курсор на карточку.
7. Нажать кнопку `В корзину`.
8. Открыть корзину.

**Ожидаемый результат:**

- Пицца из нового набора слайдов добавлена в корзину.


## TC-007: Переход на страницу подробностей выбранной пиццы из слайдера


**Шаги:**

1. Открыть главную страницу сайта.
2. Найти слайдер с пиццами.
3. Выбрать пиццу в слайдере.
4. Сохранить название выбранной пиццы.
5. Нажать на картинку выбранной пиццы.

**Ожидаемый результат:**

- Открыта страница подробностей пиццы.
- На странице подробностей отображается название той же пиццы, которая была выбрана в слайдере.

## TC-008: Выбор дополнительной опции на странице подробностей пиццы

**Шаги:**

1. Открыть страницу подробностей выбранной пиццы.
2. Найти блок дополнительных опций.
3. Выбрать дополнительную опцию, например сырный или колбасный борт.

**Ожидаемый результат:**

- Дополнительная опция выбрана.
- Выбранная опция визуально отмечена на странице.

## TC-009: Добавление пиццы с дополнительной опцией в корзину

**Шаги:**

1. Открыть страницу подробностей выбранной пиццы.
2. Выбрать дополнительную опцию.
3. Сохранить название пиццы, название опции и итоговую цену.
4. Нажать кнопку `В корзину`.
5. Открыть корзину.

**Ожидаемый результат:**

- Пицца с выбранной дополнительной опцией добавлена в корзину.
- В корзине отображается название пиццы.
- В корзине отображается выбранная дополнительная опция.
- Итоговая цена позиции соответствует цене на странице подробностей.

## TC-010: Проверка состава корзины после добавления пицц

**Шаги:**

1. Добавить в корзину несколько пицц из слайдера.
2. Добавить в корзину пиццу с дополнительной опцией со страницы подробностей.
3. Открыть корзину.

**Ожидаемый результат:**

- В корзине отображаются все добавленные пиццы.
- Для каждой позиции отображается корректное название.
- Для пиццы с дополнительной опцией отображается выбранная добавка.
- Для каждой позиции отображается корректное количество.
- Общая сумма соответствует сумме позиций.

## TC-011: Увеличение количества пиццы в корзине

**Приоритет:** High

**Шаги:**

1. Открыть корзину с несколькими товарами.
2. Выбрать одну позицию с пиццей.
3. Увеличить количество этой позиции.
4. Нажать кнопку `Обновить корзину`.

**Ожидаемый результат:**

- Количество выбранной позиции обновлено.
- Стоимость выбранной позиции пересчитана.
- Общая сумма корзины пересчитана.
- Остальные позиции не изменились.

## TC-012: Удаление пиццы с дополнительной опцией из корзины

**Приоритет:** High

**Шаги:**

1. Открыть корзину, в которой есть пицца с дополнительной опцией.
2. Найти позицию с дополнительной опцией.
3. Удалить эту позицию из корзины.

**Ожидаемый результат:**

- Позиция с дополнительной опцией удалена из корзины.
- Остальные позиции остались в корзине.
- Общая сумма корзины пересчитана.

## TC-013: Переход в раздел десертов через меню

**Приоритет:** Medium

**Шаги:**

1. Находясь не на главной странице, открыть вкладку `Меню`.
2. В выпадающем списке выбрать раздел `Десерты`.

**Ожидаемый результат:**

- Открыт раздел `Десерты`.
- Пользователь не возвращался на главную страницу для перехода к десертам.
- На странице отображаются товары из раздела десертов.

## TC-014: Фильтрация десертов по бюджету 135 рублей

**Приоритет:** High

**Шаги:**

1. Открыть раздел `Десерты`.
2. Применить ограничение цены до `135` рублей.
3. Получить список отображаемых десертов.

**Ожидаемый результат:**

- В списке отображаются только десерты стоимостью не больше `135` рублей.
- Десерты дороже `135` рублей не отображаются.

## TC-015: Добавление десерта в корзину после фильтрации

**Приоритет:** High

**Шаги:**

1. Открыть раздел `Десерты`.
2. Ограничить цену десертов значением `135` рублей.
3. Выбрать доступный десерт из результатов.
4. Сохранить название и цену десерта.
5. Добавить десерт в корзину.
6. Открыть корзину.

**Ожидаемый результат:**

- Выбранный десерт добавлен в корзину.
- В корзине отображается название выбранного десерта.
- Цена десерта в корзине не превышает `135` рублей.
- Ранее добавленные товары остались в корзине.

## TC-016: Попытка оформить заказ неавторизованным пользователем

**Приоритет:** High

**Шаги:**

1. Открыть корзину с товарами.
2. Нажать кнопку `Оформить заказ`.

**Ожидаемый результат:**

- Сайт предлагает пользователю авторизоваться.
- Неавторизованный пользователь не может сразу перейти к оформлению заказа.

## TC-017: Регистрация нового пользователя через вкладку «Мой аккаунт»

**Приоритет:** High

**Шаги:**

1. Открыть вкладку `Мой аккаунт`.
2. Нажать кнопку `Зарегистрироваться`.
3. Ввести уникальный логин.
4. Ввести уникальный email.
5. Ввести пароль.
6. Подтвердить регистрацию.

**Ожидаемый результат:**

- Регистрация завершена успешно.
- Пользователь авторизован после регистрации.
- Ошибки валидации не отображаются.

## TC-018: Проверка авторизации после регистрации

**Приоритет:** High

**Шаги:**

1. После регистрации открыть вкладку `Мой аккаунт`.
2. Проверить состояние аккаунта.

**Ожидаемый результат:**

- В личном кабинете отображается информация авторизованного пользователя.
- На странице нет формы входа/регистрации для гостя.
- Пользователь понимает, что он залогинен.

## TC-019: Переход к оформлению заказа после регистрации

**Приоритет:** High

**Шаги:**

1. Авторизоваться или зарегистрироваться.
2. Открыть корзину с товарами.
3. Нажать кнопку `Оформить заказ`.

**Ожидаемый результат:**

- Пользователь переходит на страницу оформления заказа.
- Все товары из корзины доступны в заказе.
- Общая сумма заказа соответствует сумме корзины.

## TC-020: Заполнение данных доставки

**Приоритет:** High

**Шаги:**

1. Открыть страницу оформления заказа.
2. Заполнить обязательные данные получателя.
3. Заполнить адрес доставки.
4. Заполнить контактные данные.

**Ожидаемый результат:**

- Все обязательные поля заполнены.
- Ошибки валидации не отображаются.
- Пользователь может продолжить оформление заказа.

## TC-021: Выбор даты доставки на завтра

**Приоритет:** Medium

**Шаги:**

1. Открыть страницу оформления заказа.
2. Найти поле выбора даты доставки.
3. Выбрать дату, равную завтрашнему дню.

**Ожидаемый результат:**

- В поле даты установлена завтрашняя дата.
- Сегодняшняя дата не выбрана.

## TC-022: Выбор оплаты при доставке и подтверждение заказа

**Приоритет:** High

**Шаги:**

1. Открыть страницу оформления заказа с заполненными данными доставки.
2. Выбрать способ оплаты `Оплата при доставке`.
3. Подтвердить заказ.

**Ожидаемый результат:**

- Заказ успешно оформлен.
- Пользователь видит страницу или блок подтверждения заказа.
- Выбранный способ оплаты отображается в подтверждении.

## TC-023: Проверка подтверждения заказа

**Приоритет:** High

**Шаги:**

1. Оформить заказ.
2. Открыть подтверждение заказа.
3. Проверить список товаров.
4. Проверить общую сумму.
5. Проверить личные данные пользователя.
6. Проверить данные доставки.
7. Проверить дату доставки.
8. Проверить способ оплаты.

**Ожидаемый результат:**

- В подтверждении отображаются все выбранные товары.
- Количество товаров соответствует корзине перед оформлением.
- Пицца с удалённой дополнительной опцией отсутствует.
- Десерт присутствует в заказе.
- Общая сумма корректна.
- Личные данные пользователя корректны.
- Дата доставки равна завтрашнему дню.
- Способ оплаты равен `Оплата при доставке`.

## TC-024: End-to-end сценарий основного флоу

**Приоритет:** Critical

**Шаги:**

1. Открыть главную страницу сайта.
2. Добавить несколько пицц из слайдера в корзину.
3. Переключить слайдер вправо и влево.
4. Добавить пиццу после переключения слайдера.
5. Открыть подробности одной пиццы из слайдера.
6. Выбрать дополнительную опцию.
7. Добавить пиццу с дополнительной опцией в корзину.
8. Открыть корзину.
9. Проверить состав корзины.
10. Увеличить количество одной пиццы.
11. Нажать `Обновить корзину`.
12. Удалить пиццу с дополнительной опцией.
13. Перейти через меню в раздел `Десерты`.
14. Отфильтровать десерты по цене до `135` рублей.
15. Добавить десерт в корзину.
16. Открыть корзину.
17. Нажать `Оформить заказ`.
18. Перейти в `Мой аккаунт`.
19. Зарегистрировать нового пользователя.
20. Убедиться, что пользователь авторизован.
21. Вернуться в корзину.
22. Повторно нажать `Оформить заказ`.
23. Заполнить данные доставки.
24. Выбрать дату доставки на завтра.
25. Выбрать оплату при доставке.
26. Подтвердить заказ.
27. Проверить подтверждение заказа.

**Ожидаемый результат:**

- Пользователь успешно проходит весь основной флоу.
- Корзина сохраняет выбранные товары после регистрации.
- В заказ попадают только актуальные товары из корзины.
- Удалённая позиция отсутствует в заказе.
- Десерт с ценой до `135` рублей присутствует в заказе.
- Заказ оформлен с доставкой на завтра.
- Подтверждение заказа содержит корректную сумму, товары и личные данные пользователя.
