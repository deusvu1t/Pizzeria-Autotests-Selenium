## Главная страница

### TC-01: Отображение слайдера пицц

**Preconditions:** пользователь не авторизован
**Steps:**

1. Открыть главную страницу
2. Проверить наличие слайдера
   **Expected result:**

- Слайдер пицц отображается

---

### TC-02: Отображение первых 4 слайдов

**Preconditions:** главная страница открыта
**Steps:**

1. Проверить слайды 1–4
   **Expected result:**

- Каждый слайд отображается

---

### TC-03: Hover → кнопка "В корзину"

**Preconditions:** главная страница
**Steps:**

1. Навести курсор на слайд
   **Expected result:**

- Отображается кнопка «В корзину»

---

### TC-04: Переключение слайдера

**Preconditions:** главная страница
**Steps:**

1. Запомнить текущие пиццы
2. Переключить вперёд
3. Переключить назад
   **Expected result:**

- Список меняется при переходе вперёд
- Возвращается к исходному при возврате

---

### TC-05: Добавление пиццы после прокрутки

**Preconditions:** главная страница
**Steps:**

1. Переключить слайдер
2. Добавить новую пиццу
3. Перейти в корзину
   **Expected result:**

- Пицца добавлена
- Название и цена совпадают

---

### TC-06: Добавление нескольких пицц

**Preconditions:** главная страница
**Steps:**

1. Добавить 2 пиццы
2. Перейти в корзину
   **Expected result:**

- Обе пиццы присутствуют
- Данные корректны

## Оформление заказа

### TC-15: Требование авторизации при оформлении

**Preconditions:** пользователь не авторизован
**Steps:**

1. Добавить товар
2. Перейти в корзину
3. Нажать «Оформить заказ»
   **Expected result:**

- Отображается требование авторизации

---

### TC-16: Успешное оформление заказа

**Preconditions:** пользователь авторизован
**Steps:**

1. Добавить товар
2. Перейти в корзину
3. Перейти к оформлению
4. Заполнить форму доставки
5. Выбрать оплату
6. Подтвердить заказ
   **Expected result:**

- Заказ создан
- Отображается номер заказа
- Данные совпадают
- Сумма отображается
