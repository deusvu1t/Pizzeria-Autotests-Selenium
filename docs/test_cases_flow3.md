## Бонусная программа

### TC-1: Успешное оформление бонусной карты

**Preconditions:**

- Пользователь находится на странице бонусной программы

**Steps:**

1. Перейти на страницу бонусной программы
2. Ввести имя `Андрей`
3. Ввести телефон `+79001234567`
4. Нажать кнопку «Оформить карту»
5. Подтвердить alert

**Expected result:**

- Отображается alert с текстом «заявка отправлена»
- После подтверждения отображается сообщение «Ваша карта оформлена!»

---

### TC-2: Отправка пустой формы

**Preconditions:**

- Пользователь находится на странице бонусной программы

**Steps:**

1. Нажать кнопку «Оформить карту» без заполнения полей

**Expected result:**

- Отображается ошибка:
  - «Поле "Имя" обязательно для заполнения»
  - «Поле "Телефон" обязательно для заполнения»
