# Автотесты для учебного проекта «Пиццерия»

Данный репозиторий содержит проект по автоматизации тестирования UI для учебного веб-приложения «Пиццерия». Проект выполнен в рамках финальной работы курса «Автотесты на Python. Базовая часть».

## Стек технологий

- **Язык программирования:** Python
- **Фреймворк для тестирования:** Pytest
- **Инструмент автоматизации браузера:** Selenium WebDriver
- **Отчетность:** Allure Reports
- **Статический анализ кода:** Flake8
- **CI/CD:** GitHub Actions

## Архитектура проекта

Проект построен с использованием паттерна **Page Object Model (POM)**:

- `pages/`: Классы объектов страниц, описывающие логику взаимодействия с конкретными страницами сайта.
- `components/`: Переиспользуемые компоненты (например, корзина, карточка товара, навигационная панель).
- `fixtures/`: Фикстуры Pytest для инициализации драйвера и подготовки страниц.
- `tests/`: Тестовые сценарии, разбитые по функциональным модулям.
- `utils/`: Вспомогательные инструменты (хелперы, логгер).

## Покрытие тестами

Проект включает в себя проверку следующих сценариев:

- Регистрация и личный кабинет.
- Главная страница и навигация.
- Каталог товаров и карточка продукта.
- Работа с корзиной (добавление, удаление, изменение количества).
- Применение купонов и бонусная система.
- Оформление заказа (Checkout).

## Установка и запуск

### Предварительные требования

- Python 3.10+
- Установленный браузер Chrome (тесты настроены на запуск через Chrome по умолчанию).

### Шаги по установке

1.  Клонируйте репозиторий:

    ```bash
    git clone <url_репозитория>
    cd pizzeria-autotests-selenium
    ```

2.  Создайте и активируйте виртуальное окружение:

    ```bash
    python -m venv .venv
    # Для Windows:
    .venv\Scripts\activate
    # Для macOS/Linux:
    source .venv/bin/activate
    ```

3.  Установите зависимости:
    ```bash
    pip install -r requirements.txt
    ```

### Запуск тестов

Для запуска всех тестов выполните команду:

```bash
pytest
```

Результаты тестов для Allure будут сохранены в папку `allure-results` (настроено в `pytest.ini`).

### Генерация отчета Allure

1.  Убедитесь, что у вас установлен [Allure command-line tool](https://docs.qameta.io/allure/#_installing_a_commandline).
2.  Выполните команду для генерации и открытия отчета:
    ```bash
    allure serve allure-results
    ```
